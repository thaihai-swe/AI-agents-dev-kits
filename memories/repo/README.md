# Repository Memory

This directory holds durable, project-specific memory for the AI Dev Kit.

- `constitution.md` stores repository-wide rules and guardrails.
- `project-knowledge-base.md` stores durable patterns and reusable context.
- `legacy-system-watchouts.md` stores stable brownfield gotchas.
- `architecture-decisions.md` stores inferred or confirmed architectural reasoning.
- `integration-points.md` stores reusable guidance about where and how new work connects.

Use `memories/README.md` for the top-level overview and `memories/repo/` as the canonical home for repository memory files.
