# Templates

## Purpose

Templates define the expected artifact shapes used by the workflow.

## Location

Templates live in the relevant skill under `skills/*/references/`.

## Maintainer Rules

- templates should match the current agent contracts
- templates should stay concise enough for real usage
- user docs should describe behavior-level expectations, not reproduce template internals line by line
