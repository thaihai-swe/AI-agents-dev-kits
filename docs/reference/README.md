## Quick Tables

### All 13 Commands

| Command               | Purpose           | When            |
| --------------------- | ----------------- | --------------- |
| `/spec`               | Write spec        | Start feature   |
| `/plan`               | Plan work         | After spec      |
| `/tasks`              | Create tasks      | After plan      |
| `/design`             | Technical design  | For complex     |
| `/implement`          | Write code        | During dev      |
| `/review`             | Verify quality    | After code      |
| `/analyze`            | Research          | Anytime         |
| `/discover`           | Explore code      | First time      |
| `/architecture`       | Document design   | After discovery |
| `/constitution`       | Define rules      | Project start   |
| `/patterns`           | Document patterns | After week 1    |
| `/requirement-review` | Review spec       | Optional        |
| `/archive`            | Save learnings    | After work      |

### Terms by Category

**Planning:** Spec, Design, Plan, Task, Acceptance Criteria

**Development:** Implementation, Code Review, Tests

**Quality:** Definition of Ready, Definition of Done

**Workflow:** Greenfield, Brownfield, Feature Workflow

**Memory:** User Memory, Session Memory, Repo Memory
