# Artifacts

This directory contains workflow artifacts used by the kit itself and by repositories adopting the kit.

## Canonical Feature Path

Per-feature workflow artifacts belong under:

`artifacts/features/<feature-slug>/`

Typical files include:

- `analysis.md`
- `spec.md`
- `requirements-review.md`
- `design.md`
- `plan.md`
- `tasks.md`
- `review.md`
- `testing-scenarios.md`

## Notes

In this repository, `artifacts/features/` remains the canonical destination for feature-scoped delivery artifacts.
