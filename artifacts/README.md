# Artifacts

This directory contains workflow artifacts used by the kit itself and by repositories adopting the kit.

## Canonical Feature Path

Per-feature workflow artifacts belong under:

`artifacts/features/<feature-slug>/`

Typical files include:

- `analysis.md`
- `spec.md`
- `requirements-review.md`
- `design.md`
- `plan.md`
- `tasks.md`
- `review.md`

## Other Artifact Areas In This Repository

This kit repository also includes supporting artifact areas used for repo evolution and analysis:

- `artifacts/analysis/`
- `artifacts/enhancements/`

Those directories are not the canonical destination for per-feature delivery artifacts in adopting repositories.
